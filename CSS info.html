CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages. Below are some examples of CSS code:

1. Basic Styling


body {
  font-family: Arial, sans-serif;
  background-color: #f0f0f0;
  color: #333;
}

2. Text Styling


h1 {
  color: #6A5ACD;
  text-align: center;
  font-size: 2.5em;
}

p {
  line-height: 1.6;
  font-size: 1.1em;
}

3. Box Model


.box {
  width: 200px;
  height: 200px;
  padding: 20px;
  border: 2px solid #6A5ACD;
  margin: 10px;
  background-color: #fff;
}

4. Flexbox Layout


.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.item {
  flex: 1;
  margin: 10px;
  padding: 20px;
  background-color: #f4f4f4;
}

5. Grid Layout


.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

.grid-item {
  background-color: #f4f4f4;
  padding: 20px;
  text-align: center;
}

6. Hover Effects


.button {
  background-color: #6A5ACD;
  color: #fff;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
  cursor: pointer;
}

.button:hover {
  background-color: #483D8B;
}

7. Media Queries


@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }

  .box {
    width: 100%;
    margin: 10px 0;
  }
}

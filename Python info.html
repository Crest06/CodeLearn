Python Basics

Python is a high-level, interpreted programming language known for its simplicity and readability. Below are some examples of Python code:

1. Hello, World!


print("Hello, World!")

2. Variables and Data Types


name = "Alice"
age = 25
height = 5.6
is_student = True

print(name, age, height, is_student)

3. Lists


fruits = ["apple", "banana", "cherry"]
print(fruits)
print(fruits[0])  # Access first element

4. Loops


for i in range(5):
    print(i)

# Loop through a list
for fruit in fruits:
    print(fruit)

5. Functions


def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))

6. Conditional Statements


age = 18
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

7. Dictionaries


person = {
    "name": "Alice",
    "age": 25,
    "is_student": True
}

print(person["name"])
print(person.get("age"))
